# DocInsight Support

This repository hosts the public issue tracker for [DocInsight](https://docinsight.dev).

Use this repository to:

- Report reproducible bugs
- Request features or improvements
- Share minimal reproducible examples for investigation

For general help, documentation, samples, downloads, licensing, community discussion, and private support options, visit the DocInsight support page:

<https://docinsight.dev/support>

> [!IMPORTANT]
> Public issues are visible to everyone. Do not post license keys, private source code, debug logs that contain source code or secrets, confidential documents, or customer data.
>
> Redact sensitive information before posting, or use a private support channel listed on the support page.
